#Concept:
In Playwright CSS and XPATH selectors are supported.

await  expect(page.locator('#usertype').last()).toBeChecked()
expect( await page.locator('#terms').isChecked()).toBeFalsy()
In above to assertion there is difference of wait where, In first assertion await is provided outside the expect() coz
in that toBeChecked() action is performed outside the expect block AND in second assertion
isChecked() action is written is expect block so await is provided inside it

#CSS Selector:
IF ID is present
css -> #id OR tagName#id

If class attribute is present
css -> .className  OR tagName.className

Write css on any attribute
css -> [attribute='value']

Write css with traversing from parent to child
css -> parentTagName >> childTagName
