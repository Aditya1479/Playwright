1.Create a classes respective to Pages(loginPage,DashBoardPage,OrderPage).
2. Create constructor with page as argument and call it like=> this.page=page.
3. write locators in constructor eg. this.signInButton= page.locator('#login');
4.create a methods to perform action
 async goTo(page){
        await page.goto("https://rahulshettyacademy.com/client/")
    }
5.export that class => module.exports={LoginPage};
6.create a POM manager file and create a class where constructor will have the object of
=> this.page=page
also will call the page layers like =>
=> this.loginPage = new LoginPage(page);
and create a methods to return the pages in it
 => getLoginPage() {
       return this.loginPage
    }

12.How to pass test data as fixture by extend test annotation behaviour
    12.1 create one file in utils which reads the test data
           const base = require("@playwright/test");

           exports.CustomeTest = base.extend({
           //CustomeTest will be exported and can be used in script file
               testDataOrder: {
                   username: "Aditya123@gmail.com",
                   password: "Aditya@3098",
                   productName: "ZARA COAT 3",
                   url: "https://rahulshettyacademy.com/client"
               },
           });
     12.2 add this in your script file=> const { CustomeTest} = require("../utils/test-base");
     and instead of
     test(" Client App login ", async ({page}) => {}
     use => CustomeTest(" Client App login ", async ({page,testDataOrder}) => {}
     12.3 replace the arguments and use it with testDataOrder.userName etc..
